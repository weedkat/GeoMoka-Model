# GeoMoka Model Documentation

GeoMoka Model is a geospatial semantic segmentation framework focused on remote sensing workflows.

This documentation covers:

- Project setup and installation
- End-to-end training workflow
- CLI command reference
- Codebase structure for contributors

## Highlights

- Multiple segmentation backbones via `segmentation-models-pytorch`
- DPT/DINOv2 model support
- Dataset split tooling for segmentation datasets
- Config-driven training and inference pipeline
